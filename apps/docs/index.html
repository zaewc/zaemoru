Zaemoru Framework-agnostic Web Components with adapter packages for modern UI stacks.
Start Actions Inputs Navigation Feedback Surfaces Data Commerce Adapters

One UI system. Native custom elements. Many frameworks.

Zaemoru is built around @zaemoru/elements. React, Vue, Angular, Svelte, Solid, Qwik, Lit, Preact, Astro, Alpine.js, HTMX, and Ember adapters stay thin so the product quality lives in the Web Components.

Install

Import tokens once, then register the custom elements once per app.

Plain HTML

<link rel="stylesheet" href="@zaemoru/tokens/index.css" />
<script type="module" src="@zaemoru/elements"></script>

<zm-button>Continue</zm-button>

React

import "@zaemoru/tokens/index.css";
import { Button, Modal } from "@zaemoru/react";

export function App() {
  return <Button>Continue</Button>;
}

Adapters

Adapters register the Web Components and expose framework-specific integration helpers where useful.

Vue

import "@zaemoru/tokens/index.css";
import { ZaemoruVue } from "@zaemoru/vue";

app.use(ZaemoruVue);

Angular

import "@zaemoru/tokens/index.css";
import "@zaemoru/angular";

// Add CUSTOM_ELEMENTS_SCHEMA in the Angular module/component.
