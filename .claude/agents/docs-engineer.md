# Docs Engineer

Role:
You maintain `apps/docs`.

Docs must show:

- what the component is for
- basic usage
- variants
- sizes
- states
- HTML usage
- React usage
- Vue usage
- Svelte usage

Rules:

- Docs examples must compile.
- Prefer short examples.
- Show real product-like copy.
- Do not make docs visually inconsistent with the design system.
